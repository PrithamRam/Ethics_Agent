Medical Ethics AI Assistant

An AI-powered system for analyzing medical ethics queries using literature review and GPT-4 analysis.

Prerequisites

  • Python 3.8 or higher
  • OpenAI API key (Get one here)
  • PubMed API key (Get one here)
  • Git installed on your machine

Step-by-Step Installation

1. Clone the Repository

git clone https://github.com/PrithamRam/Ethics_Agent.git
cd Ethics_Agent

2. Set Up Virtual Environment

# Create virtual environment
python -m venv .venv

# Activate virtual environment
# On Windows:
.venv\Scripts\activate
# On macOS/Linux:
source .venv/bin/activate

3. Install Dependencies

pip install -r requirements.txt

4. Configure API Keys

# Create .env file
cp .env.example .env

# Edit .env file with your API keys:
OPENAI_API_KEY=your_openai_key_here
PUBMED_EMAIL=your_email@domain.com
PUBMED_API_KEY=your_pubmed_api_key

5. Run the Application

python main.py

6. Access the Interface

Open your web browser and go to: http://localhost:8003

Common Issues & Solutions

  • ModuleNotFoundError: Make sure you've activated the virtual environment and installed requirements
  • API Key Errors: Verify your API keys in the .env file
  • Port in Use: If port 8003 is busy, modify the port in main.py
  • Database Issues: Delete ethics.db and restart to rebuild the database

Using the System

  1. Enter your medical ethics query in the text box
  2. Click "Submit Query"
  3. Review the three-part analysis:
    • AI Analysis using GPT-4
    • Relevant Papers from PubMed
    • Literature-based Analysis
